
# Semtech LR20XX HLA

This is simple but yet helpful analyzer of LR20xx (LR2021 mainly) SPI communication. 

There are 2 main features:

- detect and visualize command type
- detect and visualize status of the previously executed command.
  
## Getting started

1. Build your extension by updating the Python files for your needs
2. Create a public Github repo and push your code 
3. Update this README
4. Open the Logic app and publish your extension
5. Create a Github release
6. Debug your hardware like you've never done before :)
